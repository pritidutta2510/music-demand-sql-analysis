# Music Demand Analysis using SQL

# Project Overview
This project analyses transactional data from a music retail business to identify demand trends across genres and customer segments.

# Tools Used
SQL
Excel
Power BI

# Key Analysis
- Genre-based demand patterns
- Customer purchase behaviour
- Sales performance KPIs

# Insights
The analysis identifies high-performing genres and consumer segments, helping businesses optimise pricing, promotions and inventory planning. 
The project also serves as a practical learning resource for understanding the fundamentals of SQL, 
making it useful for educational purposes, such as placement preparation and building a strong foundation in SQL.
